# 🐵 ¡Bienvenido a MonkeyLand!

> *Un servidor de Minecraft 1.21.1 diseñado para los amigos más locos*

---

## 🎮 Acerca del Servidor

¡Hola jugador! Bienvenido a **MonkeyLand**, tu nuevo hogar virtual en Minecraft. Este es un servidor de supervivencia hard-core donde la diversión, la aventura y la amistad van de la mano.

### 🎯 Características del Servidor

- **Comunidad**: Grupo cerrado de amigos
- **Versión**: Minecraft 1.21.1
- **Modo**: Supervivencia
- **Dificulta**: Hard

---

## 📦 Mods del Servidor

### ⚠️ MODS OBLIGATORIOS
*Debes instalar estos en tu cliente para conectarte al servidor (Server-Side/Client-Server)*

| Mod | Tipo | Descripción |
|-----|------|-------------|
| **Fabric API** | Client-Server | Base requerida para que funcionen los mods |
| **Simple Voice Chat** | Client-Server | Sistema de voz integrado |
| **Waystones** | Client-Server | Viaja rápidamente entre puntos guardados |
| **Artifacts** | Client-Server | Artefactos únicos y poderosos para encontrar |
| **Decorative Blocks** | Client-Server | Bloques decorativos adicionales |
| **Advancements Enlarger** | Client-Server | Más logros para conseguir |
| **Twilight Forest** | Client-Server | Dimensión adicional para explorar |
| **Roots 3** | Client-Server | Magia natural y plantas mágicas |

> **Descarga el ModPack**, incluye todos los mods obligatorios ya configurados
> Disponible en: [Enlace al ModPack]

### ✨ MODS OPCIONALES
*Solo para tu cliente. Mejora tu experiencia personal sin afectar a otros jugadores*

| Mod | Tipo | Descripción |
|-----|------|-------------|
| **Sodium** | Client-Side | Optimización esencial del renderizado |
| **Lithium** | Client-Side | Mejoras de rendimiento de cliente |
| **Phosphor** | Client-Side | Optimización de iluminación |
| **Gravestone Mod** | Client-Side | Recupera tus items cuando mueres |
| **Minimap** | Client-Side | Mapa miniatura en la esquina de tu pantalla |

> Los mods opcionales son solo para ti. Instálalos para mejorar tu rendimiento y experiencia visual
